## Purpose

This folder contains placeholder artifacts generated by the gap analyzer. Stubs are incomplete artifacts that need research and completion before promotion to production folders.

## Characteristics

Stub artifacts contain:
- Basic ID and type fields
- Placeholder values marked with `[GAP STUB - ...]`
- `_gap_metadata` field linking to source gap
- `editor_status: "pending"`

## Example Stub

```json
{
  "authority_id": "AUTH-AR-ACA-20-77-108",
  "authority_type": "statute",
  "citation": "[GAP STUB - citation needed]",
  "administering_body": "[GAP STUB - needs research]",
  "governs": [],
  "effects": {},
  "_gap_metadata": {
    "gap_id": "GAP-AUTH-AR-ACA-20-77-108",
    "source_artifact": "MF-AR-MEDICAID-SPECIAL-NEEDS",
    "created_at": "2026-02-10T12:00:00Z"
  },
  "editor_status": "pending"
}
```

## Completion Workflow

1. **Research**: Find authoritative source for missing artifact
2. **Fill Fields**: Replace all `[GAP STUB - ...]` values
3. **Remove Metadata**: Delete `_gap_metadata` field
4. **Update Status**: Change `editor_status` to `"accepted"`
5. **Move File**: Copy to appropriate category folder
6. **Verify**: Run gap_analyzer to confirm closure

## Usage

```powershell
# Find stubs needing completion
Get-ChildItem -Filter *.json | ForEach-Object {
    if ((Get-Content $_) -match "GAP STUB") { $_.Name }
}

# Count stubs by category
Get-ChildItem -Filter *.json | ForEach-Object {
    $s = Get-Content $_ | ConvertFrom-Json
    if ($s.flow_id) { "money_flow" }
    elseif ($s.authority_id) { "authority_reference" }
    elseif ($s.evidence_id) { "evidence_item" }
    elseif ($s.fv_id) { "field_validation" }
    else { "unknown" }
} | Group-Object | Select-Object Name, Count

# List stubs from specific source
Get-ChildItem -Filter *.json | Where-Object {
    (Get-Content $_ | ConvertFrom-Json)._gap_metadata.source_artifact -match "MEDICAID"
}
```
