# _data — Dashboard and Visualization Assets

**Status:** ACTIVE  
**Created:** 2026-02-05  
**Purpose:** Contains web-based visualization tools for the Clearlane artifact graph

---

## Overview

This folder contains the interactive dashboard and supporting data files for visualizing the Arkansas public finance artifact network. The dashboard runs in a browser and can be served statically or via the FastAPI server.

---

## Files

| File | Description |
|------|-------------|
| `index.html` | **Primary dashboard** — Interactive UI with action buttons, stats cards, and Sankey diagram |
| `dashboard_viewer.html` | Standalone Sankey visualization (static, no API integration) |
| `graph_viewer.html` | Alternate graph visualization |
| `artifacts_graph.json` | Exported linkage data with full artifact content |
| `artifacts_graph.db` | SQLite database (if generated by linkage_analyzer) |

---

## Using the Dashboard

### Via FastAPI Server (Recommended)

```powershell
cd c:\Threshold\cleanroom\artifacts\validated
python -m uvicorn server.api:app --host 0.0.0.0 --port 8000
```

Then open: http://localhost:8000

### Static Mode

Open `dashboard_viewer.html` directly in a browser. Note: Action buttons will not work without the API server.

---

## Dashboard Features

### Stats Cards
- Total Artifacts
- Total Linkages  
- Orphan Count
- Total Money Tracked

### Action Buttons
- **Run Linkage Analyzer** — Regenerates the linkage graph
- **Run Gap Analyzer** — Finds missing artifacts
- **Quick Summary** — Prints overview stats

### Sankey Diagram
Visualizes money flows from sources through intermediaries to destinations.

### Data Tables
Tabbed views for each artifact category:
- Money Flow
- Authority Reference
- Evidence Item
- Field Validation

---

## Regenerating Data

The `artifacts_graph.json` file is regenerated by running:

```powershell
python scripts/linkage_analyzer.py --export-all
```

This exports:
- All nodes with full artifact data
- All linkages between artifacts
- Category metadata
